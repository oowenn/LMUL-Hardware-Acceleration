# Synthesis Output Directory

This directory contains generated synthesis outputs. **These files are not committed to git** (see `.gitignore`).

## Contents

- **`*_yosys.log`** - Yosys synthesis logs (can be regenerated)
- **`*_sta.log`** - OpenSTA timing analysis logs (can be regenerated)
- **`*_synth.v`** - Gate-level netlists (can be regenerated)
- **`*.png`** - Visualization images (generated by notebook)

## Regenerating

All files in this directory can be regenerated by:

1. **Running synthesis:**
   ```bash
   yosys -s ../scripts/synth_lmul.ys | tee lmul_yosys.log
   yosys -s ../scripts/synth_bf16.ys | tee bf16_yosys.log
   ```

2. **Running timing analysis:**
   ```bash
   sta ../scripts/sta_lmul.tcl | tee lmul_sta.log
   sta ../scripts/sta_bf16.tcl | tee bf16_sta.log
   ```

3. **Running the notebook:**
   - `synthesis_results.ipynb` will generate all visualizations

## Why Not in Git?

- Files are large (~600KB+ and growing)
- Can be regenerated from source
- Reduces repository size
- Prevents merge conflicts on generated files

The folder structure is preserved (via `.gitkeep`) so the directory exists for local use.
